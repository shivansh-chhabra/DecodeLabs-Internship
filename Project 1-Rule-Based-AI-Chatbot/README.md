# Rule-Based AI Chatbot

## Overview
This project is a simple rule-based chatbot built using Python. It responds to predefined user inputs using if-else logic and runs in a continuous loop until the user exits.

## Features
- Handles greetings
- Responds to predefined queries
- Input sanitization using strip() and lower()
- Fallback response for unknown inputs
- Exit commands (bye, exit, quit)
- Dictionary-based knowledge base

## Technologies Used
- Python 3

## How to Run

1. Clone the repository
2. Navigate to the project folder
3. Run:

python chatbot.py

## Example

You: hello
Bot: Hello! How can I help you?

You: how are you
Bot: I'm just a chatbot, but I'm doing well!

You: bye
Bot: Goodbye! Have a great day.
